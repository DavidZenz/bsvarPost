4. **Introduction** (2-3 paragraphs):
   - Bayesian structural VARs produce full posterior distributions, not just point estimates
   - bsvarPost provides tools for formal posterior probability statements about impulse responses and CDMs
   - This vignette demonstrates: pointwise hypotheses, joint hypotheses, simultaneous credible bands, and magnitude auditing
   - Continuing the fiscal policy question: "Does fiscal spending significantly increase GDP?"
   - Show estimation code with eval=FALSE (same pattern as other vignettes)

5. **Section: Pointwise posterior probabilities** (eval=TRUE):
   - Frame the question: "What is the posterior probability that a fiscal spending shock increases GDP at horizon h?"
   - `hypothesis_irf(post, variables = 3, shocks = 1, horizon = 8, relation = ">", value = 0)` -- adjust variable/shock indices based on actual us_fiscal_lsuw colnames from 07-01-SUMMARY.md
   - Explain the output: posterior probability, number of draws satisfying the condition
   - Show for CDMs too: `hypothesis_cdm(post, variables = 3, shocks = 1, horizon = 8, relation = ">", value = 0)`
   - Economic interpretation: "X% of posterior draws show a positive GDP response to fiscal spending at 2 years"

6. **Section: Joint posterior hypotheses** (eval=TRUE):
   - Frame the question: "Is the GDP response positive at EVERY horizon from 0 to 8?"
   - `joint_hypothesis_irf(post, variables = 3, shocks = 1, horizon = 0:8, relation = ">", value = 0)`
   - Explain the difference from pointwise: joint probability is the intersection of all individual conditions
   - Show plotting: eval=FALSE for `plot_joint_hypothesis()` and `plot_hypothesis()`, with the hypothesis-showcase figure included via `knitr::include_graphics("figures/hypothesis-showcase.png")`

7. **Section: Simultaneous credible bands** (eval=TRUE):
   - Frame the question: "What credible band contains the entire impulse response path with 90% posterior coverage?"
   - `simultaneous_irf(post, horizon = 20, variables = 3, shocks = 1)`
   - Explain the difference from pointwise credible intervals: simultaneous bands are wider but cover the whole function
   - `simultaneous_cdm(post, horizon = 20, variables = 3, shocks = 1)`
   - Economic interpretation: the entire fiscal multiplier path lies within these bands

8. **Section: Magnitude auditing** (eval=TRUE):
   - Frame the question: "What fraction of posterior draws show a GDP multiplier exceeding 1.0?"
   - `magnitude_audit(post, type = "cdm", variables = 3, shocks = 1, horizon = 8, relation = ">", value = 1)`
   - Explain: this is a post-estimation summary, not an identification constraint
   - Show comparison: `magnitude_audit(post_alt, type = "cdm", variables = 3, shocks = 1, horizon = 8, relation = ">", value = 1)` -- compare baseline vs alternative

9. **Section: Comparing hypothesis results across models** (eval=TRUE):
   - Compare pointwise probabilities across specifications
   - `hypothesis_irf(post, ...)` vs `hypothesis_irf(post_alt, ...)` side by side
   - Tabulate with `as_kable()` for comparison
   - Economic insight: does lag specification affect the evidence for fiscal stimulus effectiveness?

10. **Section: Summary** (brief):
    - Recap the hypothesis testing toolkit
    - Note that these are proper Bayesian posterior probability statements, not frequentist p-values
